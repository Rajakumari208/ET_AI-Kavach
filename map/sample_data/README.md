# Sample Data — Synthetic Only

`incidents.csv` in this folder is **synthetically generated** for prototype
demonstration purposes. It does not represent real crime reports, real
victims, or real locations tied to any actual case.

It was generated by a small script (see `/home/claude/kavach/map/sample_data`
generation notes in the main repository `data/DATA_SOURCES.md`) that
randomly jitters coordinates around five Indian metro areas and assigns
random incident types, severities, and timestamps within a synthetic date
window.

**Production deployment note:** the Map dashboard (`map/dashboard/app.py`)
is built to ingest a CSV/API feed with the same column schema
(`incident_id, city, latitude, longitude, incident_type, severity,
reported_at`) from an authorized source such as NCRB, a state cybercrime
cell, or the 1930 helpline reporting system — this sample file exists only
so the dashboard is demoable without that live integration.
